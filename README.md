# jwp-shopping-order

## 기능 요구 목록
- [x] 주문 API 구현
  - [x] 쿠폰이 있으면 쿠폰 삭제도 해야한다.
  - [x] 상품이 있는지 없는지 검증한다.
  - [x] 사용된 쿠폰을 다시 사용 못하도록 검증한다.
  - [x] 쿠폰을 사용하면 사용처리를 해야한다.
- [x] 주문 목록 조회 API 구현
- [x] 전체 쿠폰 조회 API 구현
- [x] 사용자 쿠폰 조회 API 구현
- [x] 사용자 쿠폰 삭제 API 구현
  - [x] 사용 안 된 쿠폰은 삭제 못하게 한다.
- [x] 사용자 쿠폰 발급 API 구현
  - [x] 이미 있는 쿠폰은 발급하지 못하게 한다.
- [x] 쿠폰 상태 변경 API 구현
  - [x] 사용 안 된 쿠폰은 재발급 하지 못하게 한다.
